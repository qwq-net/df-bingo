# ドルフロ ビンゴシミュレーター:Re

2020年頃、Vue.jsを使用して作成したドルフロのビンゴシミュレーターをReactで書き直したプロジェクトです。

## 開発の始め方

1. **依存関係のインストール:**

   ```bash
   npm install
   ```

2. **開発サーバーの起動:**

   ```bash
   npm run dev
   ```

3. **ビルド:**
   ```bash
   npm run build
   ```

詳細は [docs/setup.md](file:///home/intn/repositories/df-bingo/docs/setup.md) を参照してください。
